Terug

Mijn vooropleiding en mijn ervaring

Voordat ik de opleiding Applicatie-en Mediaontwikkeling ging doen zat ik op het Gemini College Lekkerkerk, niveau vmbo-tl. Ik heb niet zo genoten van de middelbareschool, maar ik wist dat als ik zou slagen dat het echte werk zou beginnen. Op het Gemini College Lekkerkerk heb ik een aantal dingen geleerd en heb ik mij goed in kunnen ontwikkelen. Ik kan super goed Engels en kan mijzelf erg goed verwoorden. Ook kan ik mij heel goed verwoorden in het Nederlands. Dit kan ik nu ontzettend goed toepassen op mijn huidige opleiding.

Mijn bijbaantjes en vooropleiding

Ik heb twee bijbaantjes gehad. Toen ik vijftien jaar oud was begon ik als vakkenvuller bij de Coop in Alblasserdam. Hier leerde ik zelfstandig te werken en was ik een aanspreekpunt voor de klanten. Toen ik zestien jaar oud was nam ik ontslag bij de Coop en begon ik als crewlid bij McDonald’s Alblasserdam. Ik ben in één jaar tijd getraind om alles in de keuken te kunnen. Van friet bakken tot burgers opdienen. Wat ik bij de Coop heb geleerd is om zelfstandig te werken. Toen bij de McDonald’s ging werken leerde ik ook heel goed om onder hoge druk te werken. Het zelfstandig werken kan ik goed toepassen aan deze opleiding, maar werken onder hoge druk weet ik nog net niet. Als je namelijk haastig te werk gaat met programmeren maak je veel fouten, maar het helpt je ook voor later. De klant geeft je bijvoorbeeld een deadline en dan moet je wel klaar zijn. Dus ik kan goed functioneren onder hoge druk. Op de middelbare school heb ik dus geleerd hoe ik mij goed kan verwoorden en ik kan perfect Engels. Aangezien er heel veel informatie van code in het Engels op het internet staat is dit heel handig toe te passen. Ik kan ook erg goed gesprekken voeren omdat ik mij erg goed kan verwoorden. Dit zal zeker van pas komen in de zakenwereld. Ook voor de communicatie is dit natuurlijk erg handig.
