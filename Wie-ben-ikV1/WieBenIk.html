Hoe blijf ik gemotiveerd?

Ik blijf gemotiveerd om tegen mezelf te zeggen dat ik dit voor mijzelf doe. Mijn toekomst, mijn keuzes. Ik wil hierin slagen. Ik weet dat er tegenslagen zullen komen, maar van deze tegenslagen zal ik juist leren.

Wat doe ik buiten de lessen aan programmeren?

Als ik vrijetijd heb probeer ik zoveel mogelijk te programmeren om mijzelf zelfstandiger te maken. Ik kijk bijvoorbeeld naar tutorials op YouTube en ik kijk naar projecten van andere programmers om inspiratie op te doen of om er van te leren.

Hoe wil ik mij ontwikkelen?

Ik wil mijzelf natuurlijk gaan ontwikkelen tot een super programmer. Ik wil zelfstanding te werk kunnen gaan. Ik wil problemen kunnen vinden en kunnen oplossen in mijn code, maar dat wil iedereen natuurlijk. Ik wil code kunnen lezen en kunnen begrijpen. Hier wil ik mij in ontwikkelen voor dit jaar.
